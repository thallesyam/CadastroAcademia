# CadastroAcademia#Sistema de academia

CRUD de academia feito em:

- HTML
- CSS
- JavaScript
- Node JS

Utilizei o modulo express e a view engine Nunjucks para fazer o dinamismo da página.

Consumindo API do unsplash para geração de imagens
